1. AC Basics

Fig 15.1

Alternating Current and Voltage

An alternating current (AC) is one that varies sinusoidally with time, periodically reversing direction. Unlike direct current (DC) which flows in one direction, AC continuously oscillates. Most electricity supplied to homes and industries is AC.

$$V = V_0\sin(\omega t), \quad I = I_0\sin(\omega t + \phi)$$ $$\omega = 2\pi f$$

Peak, RMS, and Average Values

Since AC varies continuously, we need different ways to characterize its magnitude:

$$V_{\text{rms}} = \frac{V_0}{\sqrt{2}} \approx 0.707 V_0$$ $$I_{\text{rms}} = \frac{I_0}{\sqrt{2}} \approx 0.707 I_0$$ $$V_{\text{avg}} = \frac{2V_0}{\pi} \approx 0.637 V_0 \quad \text{(half cycle)}$$

The RMS value is the most important measure of AC because it gives the equivalent DC value that would produce the same heating effect. When we say "230 V AC supply," we mean Vrms = 230 V, so V0 = 230√2 ≈ 325.3 V. Similarly, for Indian household supply at 50 Hz, ω = 100π rad/s.

2. AC Through Pure R, L, and C

Fig 15.2

AC Through a Pure Resistor

When an AC voltage V = V0 sin(ωt) is applied across a pure resistance R, the current is:

$$I = \frac{V_0}{R}\sin(\omega t), \quad \phi = 0$$

AC Through a Pure Inductor

When an AC voltage is applied across a pure inductance L, the back-EMF opposes the change in current. The result is:

$$I = \frac{V_0}{\omega L}\sin(\omega t - \pi/2)$$ $$X_L = \omega L = 2\pi f L \quad \text{(inductive reactance)}$$ $$\text{Current lags voltage by } 90Â°$$

Inductive reactance increases with frequency: at high frequencies, an inductor strongly opposes AC. At DC (f = 0), XL = 0 and the inductor behaves like a short circuit.

AC Through a Pure Capacitor

When an AC voltage is applied across a pure capacitance C:

$$I = V_0 \omega C \sin(\omega t + \pi/2)$$ $$X_C = \frac{1}{\omega C} = \frac{1}{2\pi f C} \quad \text{(capacitive reactance)}$$ $$\text{Current leads voltage by } 90Â°$$

Capacitive reactance decreases with frequency: at high frequencies, a capacitor easily passes AC. At DC (f = 0), XC = ∞ and the capacitor blocks current completely (open circuit).

A useful mnemonic: "CIVIL" — in a Capacitor, I leads V; in an inductor (L), V leads I.

3. Phasor Diagrams and AC Circuit Analysis

Fig 15.3

What is a Phasor?

A phasor is a rotating vector used to represent a sinusoidally varying quantity. The length of the phasor represents the RMS amplitude of the quantity, and the angle it makes with a reference direction represents the phase. As time progresses, the phasor rotates counterclockwise at angular frequency ω, and the instantaneous value of the quantity is the projection of the phasor onto the vertical (or horizontal) axis.

All values shown in phasor diagrams are RMS values. Phasor addition follows the parallelogram (vector addition) law, which greatly simplifies analysis of AC circuits.

Phase Relationships in Basic Elements

  • Resistor: VR and I are in phase (same direction in phasor diagram). VR = IR.
  • Inductor: VL leads I by 90° (VL phasor is 90° ahead of I phasor). VL = IXL.
  • Capacitor: VC lags I by 90° (VC phasor is 90° behind I phasor). VC = IXC.

Phasor Diagram for Series RC Circuit

In a series RC circuit, the current I is common to both elements. Taking I as the reference phasor (along the positive x-axis):

  • VR is along the same direction as I (0°).
  • VC points downward, perpendicular to I (−90°).
  • The resultant voltage V = √(VR² + VC²) lags the current by angle φ = tan−1(VC/VR) = tan−1(XC/R).
$$Z = \sqrt{R^2 + X_C^2}$$ $$\phi = \tan^{-1}\left(\frac{X_C}{R}\right)$$

Phasor Diagram for Series RL Circuit

In a series RL circuit, taking I as the reference phasor:

  • VR is along I (0°).
  • VL points upward, perpendicular to I (+90°).
  • The resultant voltage V leads the current by angle φ = tan−1(VL/VR) = tan−1(XL/R).
$$Z = \sqrt{R^2 + X_L^2}$$ $$\phi = \tan^{-1}\left(\frac{X_L}{R}\right)$$

Phasor Diagram for Series LCR Circuit

In a series LCR circuit, VL and VC are exactly 180° apart and partially cancel each other. The resultant reactive voltage is |VL − VC|, and the total voltage is:

$$V = \sqrt{V_R^2 + (V_L - V_C)^2}$$ $$\phi = \tan^{-1}\left[\frac{X_L - X_C}{R}\right]$$
Example 1 — Phasor Analysis of a Series RC Circuit

An RC series circuit has Vrms = 141.5 V, VR = 81.2 V, and VC = 116 V. Verify the source voltage using phasor addition and find the phase angle between V and I.

Solution:
Taking current I as the reference phasor:
• VR = 81.2 V is along I (0°)
• VC = 116 V lags I by 90° (points downward)

Resultant source voltage (phasor sum):
V = √(VR² + VC²) = √(81.2² + 116²)
V = √(6593.4 + 13456) = √20049.4 ≈ 141.6 V   (matches given 141.5 V)

Phase angle:
φ = tan−1(VC/VR) = tan−1(116/81.2) = tan−1(1.429) ≈ 55°

Conclusion: The source voltage lags the current by 55° (capacitive circuit).

4. Series LCR Circuit

Fig 15.4

Impedance

In a series LCR circuit (R, L, and C connected in series) driven by an AC source, the same current flows through all three elements. The voltage drops across each element are:

  • VR = IR (in phase with I)
  • VL = IXL (leads I by 90°)
  • VC = IXC (lags I by 90°)

Since VL and VC are 180° apart, the net reactive voltage is |VL − VC|. The total voltage (by phasor addition) is:

V = √[VR² + (VL − VC)²] = I × Z

Impedance: Z = √[R² + (XL − XC)²]

Z = √[R² + (ωL − 1/ωC)²]
SI unit: Ohm (Ω)

Phase Angle

The phase angle φ between the applied voltage and the current is:

tanφ = (XL − XC) / R = (ωL − 1/ωC) / R
  • If XL > XC: φ > 0, circuit is inductive (voltage leads current).
  • If XC > XL: φ < 0, circuit is capacitive (current leads voltage).
  • If XL = XC: φ = 0, circuit is at resonance (purely resistive).
Example 2 — Series LCR Impedance and Phase Angle

A series LCR circuit has R = 100 Ω, L = 0.5 H, and C = 10 μF. It is connected to a 200 V, 50 Hz AC source. Find the impedance, current, and phase angle. Also find the voltage across each element.

Solution:
R = 100 Ω, L = 0.5 H, C = 10 × 10−6 F, Vrms = 200 V, f = 50 Hz
ω = 2πf = 100π rad/s

$$X_L = 100\pi \times 0.5 \approx 157.08 \text{ Î©}$$ $$X_C = \frac{1}{100\pi \times 10^{-5}} \approx 318.31 \text{ Î©}$$ $$Z = \sqrt{100^2 + 161.23^2} \approx 189.99 \text{ Î©}$$ $$I_{\text{rms}} = \frac{200}{189.99} \approx 1.053 \text{ A}$$ $$\phi = \tan^{-1}(-1.6123) \approx -58.2Â°$$ $$V_R = 105.3 \text{ V}, \quad V_L = 165.4 \text{ V}, \quad V_C = 335.2 \text{ V}$$

5. Resonance in Series LCR Circuit

Fig 15.5

Resonance Condition

Resonance occurs when the inductive reactance equals the capacitive reactance: XL = XC. At this condition, the impedance is minimum (Z = R), the current is maximum (I = V/R), and the circuit behaves as a purely resistive circuit.

$$\omega_0 = \frac{1}{\sqrt{LC}}$$ $$f_{\text{res}} = \frac{1}{2\pi\sqrt{LC}}$$ $$Z = R, \quad I_{\max} = \frac{V}{R}, \quad \phi = 0$$

Quality Factor (Q-factor)

The quality factor measures the sharpness of resonance. A high Q-factor means a narrow, sharp resonance peak (very selective circuit), while a low Q-factor means a broad, flat resonance (less selective).

$$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 CR} = \frac{1}{R}\sqrt{\frac{L}{C}}$$ $$Q = \frac{V_L}{V} = \frac{V_C}{V} \quad \text{(at resonance)}$$

At resonance, the voltage across L or C can be Q times the source voltage. This is called voltage magnification. For high-Q circuits, this can be dangerously large.

Bandwidth and Sharpness

The bandwidth of a resonant circuit is the range of frequencies over which the current is at least 1/√2 of its maximum value (half-power bandwidth):

$$\Delta\omega = \frac{R}{L}, \quad \Delta f = \frac{R}{2\pi L}$$ $$Q = \frac{\omega_0}{\Delta\omega} = \frac{f_0}{\Delta f}$$

A high Q-factor implies a narrow bandwidth (sharp resonance), which is desirable in radio tuning circuits for selecting a specific station.

6. Power in AC Circuits

Fig 15.6

Instantaneous and Average Power

The instantaneous power in an AC circuit is P(t) = V(t) × I(t). Since both V and I are sinusoidal, the power fluctuates. The quantity of practical interest is the average power over one complete cycle:

$$P_{\text{avg}} = V_{\text{rms}} I_{\text{rms}} \cos\phi$$ $$\cos\phi = \frac{R}{Z}$$

Power Factor

The power factor cosφ determines what fraction of the apparent power (VrmsIrms) is actually dissipated as useful power:

$$\cos\phi = \frac{R}{Z} = \frac{R}{\sqrt{R^2 + (X_L - X_C)^2}}$$
  • Pure resistor: φ = 0, cosφ = 1, P = VrmsIrms (maximum power dissipation).
  • Pure inductor or capacitor: φ = ±90°, cosφ = 0, P = 0 (no power dissipation).
  • At resonance: cosφ = 1, P = V²rms/R (maximum power for given source).
  • A low power factor in industrial AC circuits means higher current for the same real power, leading to greater transmission losses. Power factor correction (adding capacitors) is therefore important.

Wattless Current

The component of current that does not contribute to power dissipation is called the wattless (or reactive) current. It oscillates energy back and forth between the source and the reactive elements (L and C).

$$I_{\text{wattless}} = I_{\text{rms}}\sin\phi$$ $$I_{\text{active}} = I_{\text{rms}}\cos\phi$$ $$S = V_{\text{rms}}I_{\text{rms}} \quad (VA)$$ $$P = V_{\text{rms}}I_{\text{rms}}\cos\phi \quad (W)$$ $$Q = V_{\text{rms}}I_{\text{rms}}\sin\phi \quad (VAR)$$
Example 3 — Power and Power Factor in a Resonant LCR Circuit

A series LCR circuit with R = 50 Ω is at resonance with a 100 V RMS AC source. The quality factor is Q = 10. Find the RMS current, the power dissipated, the voltage across the inductor, and the power factor.

Solution:
At resonance: Z = R = 50 Ω, Vrms = 100 V, Q = 10

$$I_{\text{rms}} = \frac{100}{50} = 2 \text{ A}$$ $$P = \frac{V_{\text{rms}}^2}{R} = \frac{100^2}{50} = 200 \text{ W}$$ $$V_L = Q \times V_{\text{rms}} = 10 \times 100 = 1000 \text{ V}$$ $$\text{Power factor: } \cos\phi = 1$$

7. Average Power Across Individual Reactive Elements

Fig 15.7

Average Power Across a Resistor

The resistor is the only element that dissipates energy in an AC circuit. Power in the resistor is always positive (energy always flows from source to resistor):

$$P_R = I_{\text{rms}}^2 R = \frac{V_{R,\text{rms}}^2}{R}$$

Average Power Across an Ideal Inductor

In an ideal inductor, the current lags the voltage by exactly 90°. During the first quarter cycle, the inductor absorbs energy from the source (storing it in its magnetic field). During the next quarter cycle, it returns this energy back to the circuit. There is no net energy dissipation:

$$P_{L,\text{avg}} = V_{L,\text{rms}} I_{\text{rms}} \cos(90Â°) = 0 \text{ W}$$

The inductor is a purely reactive (non-dissipative) element. It temporarily stores energy in its magnetic field and releases it back, acting as an energy reservoir rather than a consumer. This is why the voltage and current being 90° out of phase makes the time-averaged power exactly zero.

Average Power Across an Ideal Capacitor

Similarly, in an ideal capacitor, the current leads the voltage by exactly 90°. During the first quarter cycle, the capacitor charges and absorbs energy (stored in its electric field). During the next quarter cycle, it discharges and returns this energy to the circuit:

$$P_{C,\text{avg}} = V_{C,\text{rms}} I_{\text{rms}} \cos(90Â°) = 0 \text{ W}$$

Like the inductor, the capacitor is a purely reactive (non-dissipative) element. It stores and releases energy cyclically with zero net dissipation over a complete cycle.

Summary: Only the resistor dissipates power (P = I²R). Ideal inductors and capacitors store and return energy with zero average power dissipation. Real inductors have some resistance, so they do dissipate a small amount of energy.

8. Parallel RC Circuit

Fig 15.8

Circuit Characteristics

In a parallel RC circuit, the same AC voltage V is applied across both the resistor and the capacitor. Each branch carries its own current, which are out of phase with each other. The total current is the phasor (vector) sum of the two branch currents.

  • Current through R: IR = V/R — in phase with V.
  • Current through C: IC = V/XC = VωC — leads V by 90°.
  • Since IR and IC are 90° apart, total current is found by Pythagoras.

Impedance and Phase Angle

$$I = \sqrt{I_R^2 + I_C^2}$$ $$\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \omega^2 C^2}$$ $$\tan\phi = \frac{I_C}{I_R} = \omega RC$$

Note: In a parallel circuit, the admittance Y = 1/Z is more convenient than impedance. The impedance of a parallel combination is always less than either branch impedance alone.

Example 4 — Parallel RC Circuit Analysis

In a parallel RC circuit connected to a 100 V, 50 Hz supply, R = 200 Ω and C = 10 μF. Find the branch currents, total current, impedance, and phase angle.

Solution:
Vrms = 100 V, f = 50 Hz, R = 200 Ω, C = 10 × 10−6 F
$$\omega = 2\pi \times 50 = 100\pi \text{ rad/s}$$ $$X_C = \frac{1}{100\pi \times 10^{-5}} \approx 318.3 \text{ Î©}$$ $$I_R = \frac{100}{200} = 0.5 \text{ A}$$ $$I_C = \frac{100}{318.3} \approx 0.314 \text{ A}$$ $$I = \sqrt{0.25 + 0.0986} \approx 0.590 \text{ A}$$ $$\phi = \tan^{-1}(0.628) \approx 32.1Â°$$

9. Parallel RL Circuit

Fig 15.9

Circuit Characteristics

In a parallel RL circuit, the same AC voltage V is applied across both the resistor and the inductor. The two branch currents are 90° out of phase with each other:

  • Current through R: IR = V/R — in phase with V.
  • Current through L: IL = V/XL = V/(ωL) — lags V by 90°.
  • The total current lags the voltage (unlike series RL, where the voltage leads the current).

Impedance and Phase Angle

$$I = \sqrt{I_R^2 + I_L^2}$$ $$\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \frac{1}{(\omega L)^2}}$$ $$\tan\phi = \frac{I_L}{I_R} = \frac{R}{\omega L}$$

Comparing with series RL: in a series RL circuit the voltage leads the current by tan−1(XL/R). In a parallel RL circuit, the current lags the voltage by tan−1(R/XL). At higher frequencies, XL increases, making R/XL smaller and the phase lag decreases.

Example 5 — Parallel RL vs. Parallel RC Comparison

A parallel RL circuit has IR = 0.8 A and IL = 0.6 A. A parallel RC circuit has IR = 0.8 A and IC = 0.6 A. Compare the total currents and phase angles in both cases.

Parallel RL: $$I_{\text{total}} = \sqrt{0.64 + 0.36} = 1.0 \text{ A}$$ $$\phi = \tan^{-1}(0.75) \approx 36.87Â° \quad \text{(lags)}$$ Parallel RC: $$I_{\text{total}} = 1.0 \text{ A}$$ $$\phi \approx 36.87Â° \quad \text{(leads)}$$

10. Parallel LCR Circuit

Fig 15.10

Circuit Characteristics

In a parallel LCR circuit, the same voltage V is applied across all three branches. Each branch draws its own current:

  • IR = V/R — in phase with V.
  • IL = V/XL — lags V by 90°.
  • IC = V/XC — leads V by 90°.

Since IL and IC are exactly 180° apart, they partially cancel. The net reactive current is |IC − IL|, and the total current is found by phasor addition with IR:

$$I = \sqrt{I_R^2 + (I_C - I_L)^2}$$ $$\tan\phi = \frac{I_C - I_L}{I_R}$$ $$\text{Resonance: } I_C = I_L, \quad I = I_R$$

Parallel Resonance

Resonance in a parallel LCR circuit occurs when IL = IC, which means 1/XL = 1/XC, i.e., XL = XC — the same condition as series resonance:

$$\omega_0 = \frac{1}{\sqrt{LC}}$$ $$Z_{\max} = R, \quad I_{\min} = I_R, \quad \phi = 0$$

This is the opposite of series resonance, where impedance is minimum and current is maximum. Parallel resonance is used in circuits where high impedance at a specific frequency is needed (e.g., tank circuits in oscillators, filters).

Note: In a parallel LCR circuit at resonance, large currents IL = IC circulate within the LC loop (tank circuit), even though the total current drawn from the source is small (just IR). This circulating current can be Q times the source current, where Q is the quality factor.

11. Transformers

Fig 15.11

Principle

A transformer works on the principle of mutual induction. It consists of two coils (primary and secondary) wound on the same soft iron core. An alternating current in the primary coil creates a changing magnetic flux in the core, which induces an alternating EMF in the secondary coil.

Transformer Equation

$$\frac{V_s}{V_p} = \frac{N_s}{N_p} = k$$ $$V_p I_p = V_s I_s$$ $$\frac{I_s}{I_p} = \frac{N_p}{N_s} = \frac{1}{k}$$

Step-Up and Step-Down Transformers

  • Step-up transformer (k > 1): Ns > Np, Vs > Vp, Is < Ip. Increases voltage, decreases current.
  • Step-down transformer (k < 1): Ns < Np, Vs < Vp, Is > Ip. Decreases voltage, increases current.

Energy Losses in Transformers

Real transformers are not 100% efficient. The main sources of energy loss are:

  • Copper losses (I²R losses): Heating of the windings due to their resistance. Minimized by using thick copper wires.
  • Iron losses (core losses): These include:
    • Eddy current losses: Eddy currents in the core dissipate energy. Minimized by using a laminated core.
    • Hysteresis losses: Energy lost in repeatedly magnetizing and demagnetizing the core. Minimized by using soft iron or special alloys with narrow hysteresis loops.
  • Flux leakage: Not all flux from the primary links with the secondary. Minimized by winding coils tightly over each other and using a closed core.
$$\eta = \frac{P_{\text{output}}}{P_{\text{input}}} = \frac{V_s I_s}{V_p I_p}$$

Long-Distance Power Transmission

Transformers are crucial for efficient long-distance power transmission. Power is transmitted at high voltage (and correspondingly low current) to minimize I²R losses in the transmission lines. At the generating station, a step-up transformer increases the voltage; at the consumer end, step-down transformers reduce it to safe levels.

$$P_{\text{loss}} = I^2 R = \left(\frac{P}{V}\right)^2 R$$

12. LC Oscillations

Fig 15.12

Principle

When a charged capacitor is connected to an inductor, the charge oscillates back and forth between the capacitor plates, producing an oscillating current. This is an electromagnetic analogue of a mechanical spring-mass system.

Charge Oscillation

If the capacitor is initially charged to Q0 and then connected to an inductor L, the charge on the capacitor at time t is:

$$q(t) = Q_0\cos(\omega t), \quad \omega = \frac{1}{\sqrt{LC}}$$ $$f = \frac{1}{2\pi\sqrt{LC}}, \quad T = 2\pi\sqrt{LC}$$

Current

$$I(t) = -Q_0\omega\sin(\omega t)$$ $$I_0 = Q_0\omega = \frac{Q_0}{\sqrt{LC}}$$

Energy Exchange

Energy continuously oscillates between the electric field of the capacitor and the magnetic field of the inductor. At any instant:

Energy in capacitor: UE = q² / (2C) = Q0² cos²(ωt) / (2C)

Energy in inductor: UB = ½LI² = Q0² sin²(ωt) / (2C)

Total energy: U = UE + UB = Q0² / (2C) = constant
  • When the capacitor is fully charged (q = Q0): all energy is in the electric field (UE = max, UB = 0, I = 0).
  • When the capacitor is fully discharged (q = 0): all energy is in the magnetic field (UE = 0, UB = max, I = I0).
  • The total energy of the system remains constant (in the ideal case with no resistance).

Analogy with Mechanical Oscillations

The LC oscillation is mathematically identical to simple harmonic motion of a spring-mass system. The correspondences are:

  • Charge q ↔ Displacement x
  • Current I ↔ Velocity v
  • Inductance L ↔ Mass m
  • 1/C ↔ Spring constant k
  • Magnetic energy ½LI² ↔ Kinetic energy ½mv²
  • Electric energy q²/(2C) ↔ Potential energy ½kx²

Damped Oscillations

In a real circuit, the resistance R of the coil and connecting wires causes energy dissipation (as I²R heating), and the oscillations gradually die out. This is analogous to a damped harmonic oscillator. The charge decays as:

q(t) = Q0 e−Rt/(2L) cos(ω't)

where ω' = √(1/LC − R²/4L²)
Example 6 — LC Oscillation: Frequency, Peak Current, and Energy

An LC circuit has L = 20 mH and C = 50 μF. The capacitor is initially charged to 100 μC. Find the resonant frequency, peak current, and the total energy of the circuit.

Solution:
L = 20 × 10−3 H, C = 50 × 10−6 F, Q0 = 100 × 10−6 C

$$\omega = \frac{1}{\sqrt{20 \times 10^{-3} \times 50 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-6}}} = 1000 \text{ rad/s}$$ $$f = \frac{1000}{2\pi} \approx 159.15 \text{ Hz}$$ $$I_0 = 100 \times 10^{-6} \times 1000 = 0.1 \text{ A}$$ $$U = \frac{(100 \times 10^{-6})^2}{2 \times 50 \times 10^{-6}} = 10^{-4} \text{ J}$$

13. Key Formulas Summary

AC Representation and Values

$$V = V_0\sin(\omega t), \quad I = I_0\sin(\omega t + \phi)$$ $$\omega = 2\pi f$$ $$V_{\text{rms}} = \frac{V_0}{\sqrt{2}}, \quad I_{\text{rms}} = \frac{I_0}{\sqrt{2}}$$

Reactances

$$X_L = \omega L \quad (\text{increases with } f)$$ $$X_C = \frac{1}{\omega C} \quad (\text{decreases with } f)$$

Series Circuits

$$Z = \sqrt{R^2 + X_C^2}, \quad \tan\phi = \frac{X_C}{R} \quad (RC)$$ $$Z = \sqrt{R^2 + X_L^2}, \quad \tan\phi = \frac{X_L}{R} \quad (RL)$$ $$Z = \sqrt{R^2 + (X_L - X_C)^2}, \quad \tan\phi = \frac{X_L - X_C}{R} \quad (LCR)$$

Parallel Circuits

$$\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \omega^2 C^2}, \quad \tan\phi = \omega RC \quad (RC)$$ $$\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \frac{1}{(\omega L)^2}}, \quad \tan\phi = \frac{R}{\omega L} \quad (RL)$$ $$\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \left(\frac{1}{X_C} - \frac{1}{X_L}\right)^2} \quad (LCR)$$

Resonance (Series and Parallel)

$$\omega_0 = \frac{1}{\sqrt{LC}}, \quad f_0 = \frac{1}{2\pi\sqrt{LC}}$$ $$Z_{\min} = R, \quad I_{\max} = \frac{V}{R} \quad (\text{series})$$ $$Z_{\max} = R, \quad I_{\min} = \frac{V}{R} \quad (\text{parallel})$$ $$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 C R}, \quad \Delta\omega = \frac{R}{L}$$

Power in AC Circuits

$$P = V_{\text{rms}}I_{\text{rms}}\cos\phi$$ $$\cos\phi = \frac{R}{Z}$$ $$P_R = I_{\text{rms}}^2 R, \quad P_L = P_C = 0$$ $$S = V_{\text{rms}}I_{\text{rms}}, \quad Q = V_{\text{rms}}I_{\text{rms}}\sin\phi$$

Transformers

$$\frac{V_s}{V_p} = \frac{N_s}{N_p} = \frac{I_p}{I_s}$$ $$V_p I_p = V_s I_s$$ $$\eta = \frac{V_s I_s}{V_p I_p}$$ $$P_{\text{loss}} = I^2 R = \left(\frac{P}{V}\right)^2 R$$

LC Oscillations

$$q(t) = Q_0\cos(\omega t), \quad \omega = \frac{1}{\sqrt{LC}}$$ $$I(t) = -Q_0\omega\sin(\omega t), \quad I_0 = \frac{Q_0}{\sqrt{LC}}$$ $$U = \frac{Q_0^2}{2C} = \frac{1}{2}LI_0^2$$ $$q(t) = Q_0 e^{-Rt/(2L)}\cos(\omega' t) \quad (\text{damped})$$
