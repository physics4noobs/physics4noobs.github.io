📐 Vector Components

Grade XI • Kinematics

About This Simulation

Resolve a vector into its rectangular components along x and y axes. Understand how any vector can be decomposed into perpendicular components. The horizontal component (cyan) and vertical component (green) together represent the original vector (purple).

Vx = V cosθ   |   Vy = V sinθ   |   V = √(Vx² + Vy²)
